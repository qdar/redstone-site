how to use the products

  1. AD - active directory (domain controller)

    The advice is to have it once you’ve overcome 20 employees

  2. Antiviruses: EDR (End Point Detection & Response), XDR (Extended Detection & Response)

    To be used per user, to be used centralized and to used on servers. They all should be WELL tuned - it’s not enough just to buy

    Well tuned anti-virus will also track SW updates which assure more security – users normally forget ignore updates

  3. Firewalls

    To be set-up between Internet and internal networks. Firewall set-up is essentially important! Firewall is nearly useless without it.

    It’s a consulting job we have deepest expertise in with all vendors.
